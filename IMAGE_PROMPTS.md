Generate each image using a **FREE** AI image tool:

| Tool | Why | Link |
|------|-----|------|
| **Microsoft Designer** (Bing Image Creator) | DALL-E 3 quality, free, commercial use OK | designer.microsoft.com |
| **Google Gemini** | Imagen, free with Google account | gemini.google.com |
| **Ideogram** | Free tier, great for marketing imagery | ideogram.ai |

## How to use this file

1. Pick one of the tools above (Microsoft Designer recommended — best quality free).
2. Copy a prompt below → paste → generate → download.
3. Save with the **exact filename** shown next to each prompt into the matching folder.
4. Reload `catalog_v2.html` — the gradient placeholder is replaced automatically.

## Style consistency

Every prompt below starts with the same **Master Style** so all images look like a cohesive collection:

> *Professional food photography, soft natural window light, warm cream and rose color palette, shallow depth of field, clean minimal marble or linen background, slight overhead angle, magazine quality, ultra-detailed, 4k*

Add: **"square 1:1 ratio"** for gallery tiles, **"landscape 4:3 ratio"** for hero/feature images.

# QUICK WORKFLOW

**Fastest path (Microsoft Designer):**
1. Go to designer.microsoft.com → sign in with Microsoft account (free)
2. Open Image Creator
3. Paste prompt → generate (4 variants per prompt)
4. Pick best → Download
5. Rename to exact filename above → save to correct folder
6. Repeat. Allow ~30 sec per image.

**Total images: ~60.** Budget ~1 hour for the full set, or batch the hero images first (6 images, 5 min) for a quick preview.

**Pro tip:** Generate the **6 hero images first** — these dominate the page. Galleries can be filled in later; the gradient placeholders look fine in the meantime.
